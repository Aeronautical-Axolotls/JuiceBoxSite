About

JuiceBox is a 2D PIC/FLIP fluid simulator built using Rust and Bevy.

Created by Team Aeronautical Axolotls!



JuiceBox is a classroom aid designed to get students engaged in discussions about fluid dynamics. Many fluid simulation tools assume that the user has some background in computational fluid dynamics (CFD), while JuiceBox does not.


The user can interact with the simulation by employing various easy-to-use tools, including the faucet tool, drain tool, wall tool, and many more!


For a more technical dive into JuiceBox, read the docs!
